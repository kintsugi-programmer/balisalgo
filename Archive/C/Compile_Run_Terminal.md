# Compile the Program

#### For C:
```bash
gcc -o hello hello.c
```

#### For C++:
```bash
g++ -o hello hello.cpp
```

The `-o` flag specifies the output file (`hello` in this case).

### Step 6: Run the Compiled Program

Run the compiled program using the following command:

```bash
./hello
```
for retarted windows

```bash
hello.exe
```
